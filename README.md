# TCBSD-in-VirtualBox
Running a virtual TwinCAT/BSD in VirtualBox

Used hardware and software:

  * HP-Z210 Workstation
  * Ubuntu 22.04.4 LTS
  * Oracle VirtualBox 6.1.50_Ubuntu r161033
  * TwinCAT 3 XAE / XAR / BSD
  * Microsoft Visual Studio 2019
